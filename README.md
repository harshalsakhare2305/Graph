# 📊 Graph — LeetCode Solutions

A curated collection of **Graph algorithm problems** solved in **Java** (with some C++), sourced from LeetCode. This repository covers a wide range of graph techniques — from classic BFS/DFS traversal to advanced topics like topological sort, Union-Find, and game theory on graphs.

## 🔑 Topics & Techniques

- **BFS (Breadth-First Search)** — shortest path, multi-source spreading, state-space exploration
- **DFS (Depth-First Search)** — connected components, cycle detection, path finding
- **Topological Sort** — dependency resolution, DAG ordering, course scheduling
- **Union-Find (DSU)** — dynamic connectivity, cycle detection, minimum spanning trees
- **Bitmask + BFS/DP** — shortest path over subsets, visiting all nodes
- **Functional Graphs** — problems where every node has exactly one outgoing edge
- **Game Theory on Graphs** — minimax-style BFS (Cat and Mouse)
- **Binary Search + Graph** — combining binary search with BFS/DFS for feasibility
- **Memoization on Graphs** — DFS with caching for DAG-based DP problems
